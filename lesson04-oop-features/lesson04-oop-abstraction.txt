4. Tính chất trừu tượng(abstraction)
Định nghĩa: 
--> Tính chất trừu tượng đc thể hiện qua các
+ class: là khái niệm trừu tượng dùng để chỉ tập hợp các đối tượng có chung thuộc tính
+ abstract_class/interface: chứa các hàm trừu tượng chứa idea có chung, sau đó sẽ đc override sau

>~ JDK 1.8
+ Functional Interface

Interface: abstract methods ~~ normal methods(default, static, private)
--> Để khởi tạo 1 đối tượng cho interface: 
    ==> Implementation Class / Anonymous class

Functional Interface: Là interface có duy nhất 1 hàm trừu tượng
--> Để khởi tạo 1 đối tượng cho interface: 
    ==> Implementation Class / Anonymous class / Anonymous function(lambda expression)
