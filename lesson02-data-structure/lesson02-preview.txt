Kiểu dữ liệu

--> là khái niệm dùng để ràng buộc, cho phép việc lưu trữ dữ liệu trong phạm vi nào
--> Ví dụ lưu chuỗi --> cần sử dụng KDL chuỗi
-->		  lưu số	--> cần sử dụng KDL số

Trong lập trình có 2 loại kiểu dữ liệu

+ static data type
--> khi có giá trị gì cần lưu chính xác trong KDL gì đó
vd: chuỗi --> String
	số	  --> int

+ dynamic(loosely) data type
--> ko quan tâm đến KDL lúc khai báo, dù là bất kỳ
giá trị lưu đc hết
--> var/let x = "Hello"; x = 5;
