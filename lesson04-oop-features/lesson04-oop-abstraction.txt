4. Tính chất trừu tượng (abstraction)
Định nghĩa:
	--> Tính chất trừu tượng được thể hiện qua các
	+ class: là khái niệm trừu tượng dùng để chỉ tập hợp các đối tượng có chung thuộc tính, phương thức
	+ abstract_class/interface: chứa các hàm trừu tượng chứa idea có chung sau đó sẽ được override
	
>~ JDK 1.8
+ Functional Interface
+ Strategy Pattern

Interface: abstract methods ~~ normal methods( default, static, private)
-->Để khởi tạo đối tượng cho interface: 
	==> Implementation Class/ Anonymous Class

Functional Interface: là interface có duy nhất 1 hàm trừu tượng
	==> Implementation Class / Anonymous class/ anonymous function (lambda expression)
